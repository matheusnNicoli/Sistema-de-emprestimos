Sistema de Empréstimos de Livros
Descrição

O Sistema de Empréstimos de Livros é uma aplicação web desenvolvida em ASP.NET Core MVC (C#) que permite o gerenciamento de registros de empréstimos por meio de operações completas de CRUD (Create, Read, Update e Delete). O projeto foi desenvolvido com o objetivo de fortalecer os conhecimentos iniciais na plataforma ASP.NET Core, aplicando os fundamentos do padrão arquitetural MVC e a integração com banco de dados relacional.

A aplicação possibilita o cadastro, visualização, edição e exclusão de empréstimos, garantindo a validação dos dados e o registro automático da data da última atualização.

Funcionalidades

Cadastro de novos empréstimos

Listagem de empréstimos cadastrados

Edição de registros existentes

Exclusão de empréstimos

Validação de campos obrigatórios

Registro automático da data de atualização

Interface web responsiva utilizando Bootstrap

Cada registro contém as seguintes informações:

Recebedor

Fornecedor

Livro emprestado

Data da última atualização

Objetivo

Este projeto tem como finalidade consolidar os conhecimentos fundamentais em desenvolvimento web com ASP.NET Core MVC, incluindo:

Estrutura e funcionamento do padrão Model-View-Controller

Criação e manipulação de Models

Implementação de Controllers e Views

Integração com banco de dados utilizando Entity Framework Core

Utilização de validações com Data Annotations

Implementação de operações CRUD completas

Tecnologias Utilizadas

C#
